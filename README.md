Fictional shop (named Multishop) made as web application in REST style. This project contains different technologies such as:

The business logic:
- Java 8,
- Spring 5,
- Spring Security 6,
- Hibernate 5,
- PostgreSQL 10.1 database.
- TDD: JUnit, Mockito.

The frontend:
- HTML5,
- CSS3,
- JavaScript,
- Thymyleaf 3
